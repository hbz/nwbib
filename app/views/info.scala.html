Open Data

Die Daten der Nordrhein-Westfälischen Bibliographie sind unter einer offenen Lizenz (CC0) zur freien Nachnutzung verfügbar. Das Hochschulbibliothekszentrum des Landes Nordrhein-Westfalen (hbz) bietet die Daten als Bestandteil des hbz-Verbundkatalogs über lobid-resources an. Die API-Dokumentation findet sich unter https://lobid.org/resources/api.

Für eine Einschränkung von lobid-API-Abfragen auf NWBib-Titel ist die Abfrage auf einfache Weise anzupassen, siehe dazu unsere API-Dokumentation.

Open Source

Der Webauftritt der Nordrhein-Westfälischen Bibliographie wird als Open-Source-Software auf GitHub entwickelt. Das GitHub-Projekt findet sich unter https://github.com/hbz/nwbib. Dort ist der Quellcode und ein Issue-Tracker verfügbar.

Das Projekt verwendet selbst verschiedene Open-Source-Projekte, darunter das Play Framework, Elasticsearch, jsonld-java und Apache Jena.
