Open Data

Die Daten der Nordrhein-Westfälischen Bibliographie sind unter einer offenen Lizenz (CC0) zur freien Nachnutzung verfügbar. Das Hochschulbibliothekszentrum des Landes Nordrhein-Westfalen (hbz) bietet die Daten als Bestandteil des hbz-Verbundkatalogs über lobid-resources an. In einem Blogbeitrag ist dokumentiert, wie die NWBib-Daten über die API von lobid-resources abgefragt werden können, siehe http://blog.lobid.org/2019/10/08/nwbib-at-cdv.html.

Für die NWBib-Raumsystematik nutzt und verbessert die NWBib Daten aus Wikidata. Für Details siehe Adrian Pohl (2021:) How We Built a Spatial Subject Classification Based on Wikidata. In: Code4Lib Journal, Issue 51.

Open Source

Der Webauftritt der Nordrhein-Westfälischen Bibliographie wird als Open-Source-Software auf GitHub entwickelt. Das GitHub-Projekt findet sich unter https://github.com/hbz/nwbib. Dort ist der Quellcode und ein Issue-Tracker verfügbar.

Das Projekt verwendet selbst verschiedene Open-Source-Projekte, darunter das Play Framework, Elasticsearch, jsonld-java und Apache Jena.
