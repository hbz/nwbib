# NWBibBot

In diesem Ordner ist die [aktuelle Version des Jupiter Notebooks für den NWBibBot aus [PAWS](https://hub-paws.wmcloud.org/hub/login?next=%2Fhub%2Fuser%2FNWBibBot%2Flab%2Ftree%2FaddNWBibLinksToWikipedia.ipynb) hinterlegt.

Der Bot:

    * geht die [Liste aller Wikidata-Einträge von Orten mit NWBib-ID durch und deutschem Wikipedia-Artikel](https://w.wiki/MQ$N),
    * und ergänzt einen Link auf eine Liste mit bibliographischen Titeln über den entsprechenden Ort in der [Nordrhein-Westfälische Bibliographie](https://de.wikipedia.org/wiki/Nordrhein-Westf%C3%A4lische_Bibliographie) auf der zugehörigen deutschen Wikipedia-Seite mittels [Vorlage:NWBib.de](https://de.wikipedia.org/wiki/Vorlage:NWBib.de).
